# World Cup 2026 Prediction Game

An interactive web application for company employees to predict FIFA World Cup 2026 outcomes.

## Tech Stack
- **Frontend:** React 18+, TypeScript, MUI, Chart.js, i18next
- **Backend:** Node.js 18+, Express, PostgreSQL, Sequelize/Prisma
- **Authentication:** JWT, bcrypt
- **External Data:** Live-Score API

## Quick Start
1. Ensure Node.js 18+ and PostgreSQL are installed.
2. Run `./init.sh` to install dependencies.
3. Configure environment variables in `.env` based on `.env.example`.
4. Run `npm run dev` in both `client` and `server` folders.

## Features
- Full prediction support for 104 matches.
- Automated scoring and leaderboards (individual & department).
- Visual bracket for knockout rounds.
- Admin dashboard for user and match management.
